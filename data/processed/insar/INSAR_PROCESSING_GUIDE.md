# InSAR Processing Guide

## Option 1: COMET LiCSAR (Recommended)
1. Visit: https://comet.nerc.ac.uk/COMET-LiCS-portal/
2. Search for region: (-105.0, 32.0, -104.0, 33.0)
3. Download interferograms (already processed!)
4. Place in data/raw/insar/
